## 🔐 Create an OpenAI Account & API Key

Before using CyberForge, you’ll need to create an OpenAI account and generate an API key. Follow the steps below to get started:

---

### 🧭 Step 1: Navigate to the OpenAI Platform

1. Go to [**openai.com**](https://www.openai.com)
2. Click **Log In** at the top right.
3. In the dropdown, select **API Platform**.

<img width="1686" alt="Screenshot 2025-06-16 at 2 04 35 PM" src="https://github.com/user-attachments/assets/6a0f5000-2130-4e7b-904d-723deee7aa35" />

---

### ✍️ Step 2: Sign Up

* If you **don’t** have an account, select **Sign up** and enter your email and password.
* Alternatively, sign in with a partner provider; Google, Microsoft, GitHub, or phone number.

<img width="1689" alt="Screenshot 2025-06-16 at 2 06 34 PM" src="https://github.com/user-attachments/assets/e6c95ad4-b8eb-4238-bf69-3783825f592c" />

*Continue with a partner provider*

<img width="1692" alt="Screenshot 2025-06-16 at 2 07 28 PM" src="https://github.com/user-attachments/assets/d2b9781c-0da5-43ea-9010-cf08b9f34a79" />

*Sign up with new account* 

---

### 📩 Step 3: Verify Your Email

* Check your inbox for a verification code.
* Enter the code to proceed.

<img width="1683" alt="Screenshot 2025-06-16 at 2 08 07 PM" src="https://github.com/user-attachments/assets/54f25736-80b4-465c-96e4-7852d7124591" />

---

### 🧑‍💼 Step 4: Enter Profile Information

* Provide your **full name** and **birth date**.

<img width="1703" alt="Screenshot 2025-06-16 at 4 19 31 PM" src="https://github.com/user-attachments/assets/37fa5be4-f90a-45c2-bdcd-f21f989929b4" />

* Add an **organization name** (can be anything).
* Select an option for “**What best describes you?**”

<img width="1705" alt="Screenshot 2025-06-16 at 4 20 05 PM" src="https://github.com/user-attachments/assets/95e01463-fb18-46cc-aec6-3773f5cd62e0" />

* Click **Create organization**.
* When prompted to invite your team, select **“Do this later”**.
  
<img width="1698" alt="Screenshot 2025-06-17 at 3 09 59 PM" src="https://github.com/user-attachments/assets/e2c02d6a-8ef4-4baf-a052-882160e854b0" />

---

### 🔑 Step 5: Generate Your API Key

1. Give the API key a name, and project name. Afterwards, click **Generate API Key**.

<img width="1704" alt="Screenshot 2025-06-16 at 4 20 39 PM" src="https://github.com/user-attachments/assets/eecab04c-d197-4a97-aa78-ed32770fafd9" />


> 📋 **Important:** Copy the API key and store it securely. You won’t be able to view it again once the dialog closes.


<img width="1703" alt="Screenshot 2025-06-16 at 4 21 42 PM" src="https://github.com/user-attachments/assets/b1224581-33aa-4b29-9638-7f941de171ee" />

---

### 💳 Step 6: Add Billing Details

* You can now purchase credits to begin using the API key, where each invokation of the API key will use your credits.

<img width="1703" alt="Screenshot 2025-06-16 at 4 23 32 PM" src="https://github.com/user-attachments/assets/917a8513-0079-4865-b133-e586705ffffe" />

#### To manage billing details and to add payment options for further usage of the API key:

1. Navigate to **Settings**.

<img width="1694" alt="Screenshot 2025-06-16 at 4 51 05 PM" src="https://github.com/user-attachments/assets/ca28b690-350e-4c55-abc6-2cc158ad30a2" />

2. Click the **Billing** tab.

<img width="1538" alt="Screenshot 2025-06-16 at 4 51 20 PM" src="https://github.com/user-attachments/assets/cd7ff54a-f32e-4583-8c42-86dcb88f3b1c" />

3. Click the green **Add payment details** button to input the credit card that will be charged for each subsequent API key invokation.

<img width="483" alt="Screenshot 2025-06-17 at 2 47 30 PM" src="https://github.com/user-attachments/assets/4ac65393-c859-4301-ba58-ffb9e9374bd8" />

### ✅ You're All Set!

You can now use your OpenAI API key in CyberForge and take full advantage of all features!
